# Pre-deploy polish — design (2026-07-20)

เอกสารนี้คือสิ่งที่ตกลงกับเจ้าของก่อน deploy รอบแรก
กลุ่มเป้าหมายของทุกการตัดสินใจ: **HR บริษัทเกมญี่ปุ่น** ที่เปิดลิงก์จาก LinkedIn/อีเมล
มีเวลาดูไม่เกิน 2–3 นาที และอาจไม่เล่นเกมเป็น

## เป้าหมาย

1. คนที่เปิดลิงก์มา **ต้องไม่หลุด** ในสามวินาทีแรก (รู้ว่าเป็นใคร สายอะไร ทำอะไรต่อได้)
2. คนที่ไม่อยากเล่นเกม **ต้องอ่านจบได้ใน Resume Mode** โดยไม่ต้องเลื่อนหาอะไร
3. เนื้อหาต้องตอบคำถามที่ HR ญี่ปุ่นถามเสมอ **ก่อนที่เขาจะต้องถาม**

### B. เนื้อหาใหม่จากเจ้าของ (ครบ 3 ภาษา)

- **เคยทำงานที่ญี่ปุ่นจริง 3 เดือน** — Tokyo Internship · บริษัท Zeal Team · ย่าน 浜松町
  (Hamamatsuchō) · พักแถว 高輪ゲートウェイ (Takanawa Gateway)
- **พร้อมย้ายไปประจำที่ญี่ปุ่น**
- โซน Freelance: เกม 2 · มังงะ 1 · งานล่าม 3+

★ จุดขายที่ตั้งใจสื่อ: ไม่ใช่แค่ *อยาก* ไปญี่ปุ่น แต่ **เคยใช้ชีวิตทำงานที่โตเกียวมาแล้ว**
— ตัดความกังวลอันดับหนึ่งของ HR ญี่ปุ่นเรื่องจ้างคนต่างชาติแล้วปรับตัวไม่ได้

### N. ไอเดียเพิ่ม (อนุมัติครบ)

- **N1** Resume Mode แบ่งเป็นกลุ่มหัวข้อ — ฉบับ ja ใช้โครง 職務経歴書 จริง
  (職務要約 → 職務経歴 → 活かせる経験・知識・技術 → 学歴 → コンタクト)
  ★ ผลพลอยได้: th/en ก็ได้กลุ่มหัวข้อไปด้วย จากเดิมเป็นการ์ดเรียง 14 ใบรวด
- **N2** กล่อง "ข้อมูลที่ HR ถามเสมอ" บนสุดของ Resume Mode
- **N3** การ์ด CTA ตอนสำรวจครบ 14 โซน → ปุ่มอีเมล/LinkedIn
- **N4** `@media print` ให้ Resume Mode พิมพ์ออกมาสะอาด
- **N5** focus trap ใน panel/Resume (a11y)
- **N6** บรรทัด "เว็บนี้เขียนเอง Vanilla JS + Canvas" — ตัวเว็บคือผลงานอีกชิ้น

## ที่ไม่ทำ (YAGNI)

- pathfinding A* เต็มรูปแบบ — แค่หลบ 1 ระดับพอ ห้องโล่งอยู่แล้ว
- analytics / tracking — ไม่มี backend และเป็นเรื่องความเป็นส่วนตัว
- ตัวอย่างงานแปล before/after — เจ้าของตัดสินใจไม่ทำรอบนี้

## โครงสร้างที่เปลี่ยน

- `resume.groups` ใน `content.js` แทน `SECTION_ORDER` ที่ hardcode ใน `resume.js`
  → ลำดับ/หัวข้อกลุ่มเป็นข้อมูล ไม่ใช่โค้ด และต่างกันได้ตามภาษา (คือหัวใจของ N1)
- `resume.facts` / `resume.summary` / `resume.cta` / `ui.hint*` — string ใหม่ครบ 3 ภาษา
- `panels.js` เพิ่ม focus trap + ตัวจับ scroll ของ `#panel-scroll`
- `renderer.js` เงาวัตถุเคลื่อนที่คิดจากหน้าต่างที่ใกล้ที่สุด (สูตรเดียวกับเงาที่อบในภาพ)
